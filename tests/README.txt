Тесты доменов

params.php
    Настройки теста. Заполнить префикс domain_prefix. Будут генерироваться
    домены вида <префикс><номер>.

mass_create.php
    Сгенерировать домены. Указывается количество доменов, которое необходимо
    создать, например
        php mass_create.php 10
    Если уже были созданы домены с таким префиксом, они продолжат создаваться
    с новыми номерами.

mass_delete.php
    Удаляет все домены с префиксом. Нужно использовать опцию -f для подтверждения
    удаления.
        php mass_delete.php -f

gen_data_set_config.php
    Генерирует файл domain.csv на основе конфигов доменов. csv-файл используется
    в jmeter-тесте.

testplan.jmx
    Jmeter-тест.
